# theme
Pufferpanel Theme

# How to Install 
```bash
cd ~/var/www/pufferpanel && \
rm -rf theme && \
git clone https://github.com/IsMeElyn/theme.git theme && \
cd theme && \
rm -f README.md && \
[ -f favicon.png ] && mv -f favicon.png ../ && \
[ -f apple-touch-icon.png ] && mv -f apple-touch-icon.png ../```
